Block vs Inline Elements

DIVs are block elements by default

Headings are block elements by default.

Spans are inline elements by default. Links are inline elements by default.

Overriding the default behavior

Using the display property, the div is changed to an inline element.

Using the display property, the h3 is changed to an inline element.

Using the display property, the span is changed to a block element. Using the display property, the link is changed to a block element.
